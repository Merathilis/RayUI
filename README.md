## 使用

1. 下载或者 clone 这个库
2. 在复制新内容之前删除旧的 `Interface` 文件夹
3. 不要管 `.gitignore, README.md` 这些文件，只把 `Interface` 文件夹复制到你的 WOW 目录的 `addons` 文件夹下
